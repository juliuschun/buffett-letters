## 왜 만들었나

LLM은 이미 워런 버핏이 누구인지 압니다. 명언도 인용합니다. 하지만 버핏에 **대해** 아는 것과 버핏**처럼** 생각하는 것은 완전히 다릅니다.

이 repo가 그 간극을 메웁니다. 일반적인 LLM이 가진 버핏 지식은 "해자가 중요하다" 수준입니다. 이 파일을 읽은 LLM은 **어떤 해자가 어떤 조건에서 작동하는지, 버핏이 왜 그렇게 믿게 되었는지, 어떤 회사가 그것을 증명했는지, 몇 년도 서한에서 그 이야기가 나오는지**까지 알게 됩니다.

차이는 이렇습니다:

- **일반 LLM**: "이 회사는 해자가 있어 보입니다"
- **이 repo를 읽은 LLM**: "이 회사의 비용 우위는 GEICO의 1986년 사례와 구조적으로 동일합니다. 비용+스케일 조합(C1 x1.3)이 적용되며, 1996년 버핏이 설명한 선순환 구조가 작동 중입니다. 다만 2006년 신문산업 사례처럼 디지털 전환이 해자를 침식할 가능성을 주시해야 합니다. 최종 해자 점수: 72/100."

## 활용 방법

### AI에게 전달하기 — 핵심 사용법

이 파일들을 AI의 컨텍스트에 넣으세요. 많이 넣을수록 분석이 날카로워집니다:

| AI에게 줄 파일 | AI가 얻는 것 |
|---|---|
| `buffett-business-checklist.md` | 평가 프레임워크 (28개 질문 + 7개 관문 + 해자 스코어링) |
| `business-reference/` (7편) | 체크리스트 모든 개념의 근거 — 질문마다 *왜* 그걸 물어야 하는지 |
| `buffett-field-guide.md` | 실전 가이드: 어디서 데이터를 찾고, 어떤 숫자를 뽑고, 어떻게 해석하는지 |
| `case-studies/` | GEICO, See's Candies 등 실전 사례 |
| `themes/` (16편) | 49년치 패턴 인식 — AI가 테마별 깊은 맥락을 흡수 |
| `lessons/` (49편) | 연도별 교훈 — 최대 깊이의 컨텍스트 |

**최소 구성**: 체크리스트 + business-reference를 주고, 아무 회사나 평가해달라고 하면 됩니다. AI가 산업, 해자, 경영진, 리스크, 밸류에이션까지 전체 버핏식 분석을 실행합니다.

**최대 구성**: 전부 넣으세요. AI가 어떤 인간 애널리스트보다 더 구조화된 버핏 지식을 갖추게 됩니다.

## 15개 테마

1. **경쟁 해자와 지속적 우위** — 해자를 만들고, 매일 넓혀라
2. **경영진의 자질과 오너-오퍼레이터 마인드셋** — 잘 뽑고, 간섭하지 마라
3. **자본 배분** — CEO의 가장 중요한 (그리고 자주 무시되는) 임무
4. **산업 경제학** — 경기장을 선택하라; 순풍이 재능을 이긴다
5. **가격 결정력과 원가 규율** — 자본 경량화의 이상
6. **장기적 사고와 복리의 힘** — 시간은 훌륭한 기업의 친구, 평범한 기업의 적
7. **리스크 관리와 재무적 요새** — 비 오기 전에 방주를 만들어라
8. **보험 모델과 플로트의 힘** — 보유하는 동안 당신을 위해 일하는 돈
9. **인수 규율** — 적정가에 훌륭한 기업 > 좋은 가격에 그저 그런 기업
10. **정직한 회계와 투명성** — 햇빛이 최고의 소독제
11. **기업 지배구조와 인센티브 설계** — 지불하는 대로 얻는다
12. **실수와 겸손** — 유일하게 용서할 수 없는 실수는 배우지 않는 것
13. **제도적 관성** — 조직이 합리성을 파괴하는 방법
14. **밸류에이션** — 시대를 초월한 투자의 수학
15. **미국에 대한 낙관** — 장기 게임에 역행하지 마라
